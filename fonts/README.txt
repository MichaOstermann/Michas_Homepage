CodeBeats Studio nutzt systemeigene Sans-Serif- und Monospace-Schriften (z. B. Segoe UI, Helvetica Neue, Ubuntu, SFMono).

So stellen wir sicher, dass keine externen Ressourcen nachgeladen werden müssen und alle Daten lokal verbleiben.

Wenn projektspezifische Custom-Fonts erforderlich sind, können sie als WOFF2-Dateien in diesem Ordner abgelegt und in `css/styles.css` via `@font-face` eingebunden werden.
